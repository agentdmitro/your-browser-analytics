## Short description

Private, local analytics for your Chrome history: activity charts, top sites/pages, categories, sessions, and JSON export.

## Detailed description

Your Browsing Analytics is a privacy-focused Chrome extension that analyzes your browsing history locally in your browser and shows usage analytics in a popup and dashboard.

No servers. No tracking. No data collection.

All processing stays on your device.

### What it shows

- Today summary in popup: visits, unique domains, top domains, hourly sparkline.
- Dashboard analytics for a selected date range.
- Top domains and top pages by visit count.
- Hourly and daily activity charts.
- Category distribution based on built-in rules.
- Search query stats (supported search engines) and session metrics (count, average, longest).
- JSON export of analytics data.

### History tools

- Delete history for today.
- Delete history for a selected date.
- Delete specific URLs (paste a list or select from recent entries).

### Categories

- 23 built-in categories.
- Custom category rules by domain or regular expression.

### Privacy

- All analytics are computed locally in Chrome.
- No external servers are used.
- No third-party analytics.
- Browsing data is not transmitted anywhere.

### Important

- This extension requires Chrome `history` permission.
- If there is no browsing history in the selected period, the popup/dashboard will show empty states.
